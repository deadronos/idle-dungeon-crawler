# Architecture Documentation

This directory contains the Decision Records for Idle Dungeon Crawler.

We use a format inspired by Architecture Decision Records (ADRs) to document significant decisions made during the development of the game.

## Structure

- `/gamedecisions/` - Documents decisions related to gameplay, mechanics, balance, and game design.
- `/technicaldecisions/` - Documents decisions related to the technology stack, architecture, patterns, and libraries used.
