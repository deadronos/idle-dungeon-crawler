# Image Generation Prompts for Idle Dungeon Crawler

This file contains prompts you can use with AI image generators (Midjourney, DALL-E, Stable Diffusion, etc.) to create game assets. Each prompt is tuned for the game's dark fantasy / idle RPG aesthetic.

## Art Style Guidelines

- **Overall vibe:** Dark fantasy, grim but polished, muted earth tones with amber/gold accents
- **Detail level:** High detail for characters and equipment, atmospheric and moody for backgrounds
- **Color palette:** Deep slate blues, warm amber highlights, desaturated greens, bone whites
- **Mood:** Mysterious, dangerous but inviting — the dungeon feels alive

## Recommended Image Sizes

| Character Portraits | 512x512 or 1024x1024   | PNG (transparent)   |
| Enemy Sprites       | 256x256 or 512x512     | PNG (transparent)   |
| Boss Sprites        | 512x512 or 1024x1024   | PNG (transparent)   |

## Notes

- For pixel-art style, add: `"pixel art, 32-bit style, crisp pixel edges, limited but rich palette"`
- For hand-painted style, add: `"hand-painted digital art, rich textures, visible brush strokes allowed"`
- Always request transparent backgrounds for sprites and icons when possible
- The game currently uses a mix of realistic-stylized and semi-pixelated assets; maintain consistency per asset category
